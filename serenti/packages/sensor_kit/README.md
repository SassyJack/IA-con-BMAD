# Sensor Kit Package (Serenti)

Módulo encargado de la recolección pasiva de metadatos de comportamiento de forma sigilosa para Serenti. Recopila información de movilidad, actividad física y socialización sin comprometer la batería.

## Responsabilidades
- Seguimiento de ubicación (GPS/WiFi) y movilidad.
- Recolección de datos de actividad física (Pasómetro).
- Inferencia de patrones de sociabilidad mediante metadatos de comunicación.

## Dependencias Clave
- `geolocator`
- `pedometer`
