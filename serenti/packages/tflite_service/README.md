# TFLite Service Package (Serenti AI)

Módulo encargado de la inferencia local de IA mediante modelos cuantizados (TFLite) para Serenti. Realiza el análisis de comportamiento (Digital Phenotyping) de forma privada en el dispositivo.

## Responsabilidades
- Carga y ejecución del modelo MLP (Perceptrón Multicapa).
- Inferencia optimizada en Isolates dedicados.
- Transformación de datos de sensores en estados emocionales para la mascota Milo.

## Dependencias Clave
- `tflite_flutter`
